🎯 EJERCICIO 1: Productos Económicos

Objetivo: Filtrar productos por rango de precios

Enunciado:

Obtén todos los productos que cuesten menos de 20€.
Luego, obtén los productos que cuesten entre 50€ y 100€.
Finalmente, cuenta cuántos productos hay en cada categoría de precio.

-SOLUCION-

# 1.1 Productos menos de 20€
productos_baratos = env['ecommerce.producto_api'].search([
    ('price', '<', 20)
])
print("🛒 Productos menos de 20€:")
for p in productos_baratos:
    print(f"  - {p.name}: {p.price}€")

# 1.2 Productos entre 50€ y 100€
productos_medios = env['ecommerce.producto_api'].search([
    ('price', '>=', 50),
    ('price', '<=', 100)
])
print(f"\n📊 Productos entre 50€ y 100€: {len(productos_medios)}")

# 1.3 Conteo por rangos
rangos = {
    'menos_20': env['ecommerce.producto_api'].search_count([('price', '<', 20)]),
    '20_50': env['ecommerce.producto_api'].search_count([
        ('price', '>=', 20), 
        ('price', '<', 50)
    ]),
    '50_100': env['ecommerce.producto_api'].search_count([
        ('price', '>=', 50), 
        ('price', '<=', 100)
    ]),
    'mas_100': env['ecommerce.producto_api'].search_count([('price', '>', 100)])
}

print("\n📈 Conteo por rangos:")
for rango, cantidad in rangos.items():
    print(f"  {rango}: {cantidad} productos")

🎯 EJERCICIO 2: Búsqueda por Categorías

Objetivo: Filtrar productos por categorías específicas

Enunciado:
text

Busca todos los productos de las categorías "electronics" y "jewelery".
Luego, encuentra todos los productos que NO sean de la categoría "men's clothing".
Finalmente, muestra cuántos productos únicos hay por categoría.

-SOLUCION-

# 2.1 Productos de electrónica y joyería
categorias_buscar = ["electronics", "jewelery"]
productos_seleccion = env['ecommerce.producto_api'].search([
    ('category', 'in', categorias_buscar)
])

print("📦 Productos en categorías seleccionadas:")
for cat in categorias_buscar:
    count = len(productos_seleccion.filtered(lambda p: p.category == cat))
    print(f"  {cat}: {count} productos")

# 2.2 Productos que NO son ropa de hombre
no_men_clothing = env['ecommerce.producto_api'].search([
    ('category', '!=', "men's clothing")
])
print(f"\n🚫 Productos que no son ropa de hombre: {len(no_men_clothing)}")

# 2.3 Conteo por categoría
from collections import Counter
todas_categorias = env['ecommerce.producto_api'].search([]).mapped('category')
conteo_cats = Counter(todas_categorias)

print("\n📊 Productos por categoría:")
for cat, count in conteo_cats.items():
    print(f"  {cat}: {count}")

🎯 EJERCICIO 3: Búsqueda por Texto

Busca productos que contengan la palabra "watch" en el nombre.
Busca productos que mencionen "cotton" en la descripción.
Busca productos que tengan "gold" en el nombre O en la descripción.

-SOLUCION-

for w in watch: print(f"{w.name}")

# Busca productos que mencionen "cotton" en la descripción.
cotton = self.env['ecommerce.producto_api'].search([('description', 'ilike', 'cotton')])
print("Busca productos que mencionen "cotton" en la descripción: ")
for c in cotton: print(f"{c.name}")

# Busca productos que tengan "gold" en el nombre O en la descripción.
gold = self.env['ecommerce.producto_api'].search(['|', ('name', 'ilike', 'gold'), ('description', 'ilike', 'gold') ])
print("Busca productos que tengan "gold" en el nombre O en la descripción: ")
for g in gold: print(f"{g.name}")


🎯 EJERCICIO 4: Productos con/sin Imagen

Objetivo: Filtrar por campos vacíos

Enunciado:

Encuentra todos los productos que NO tienen imagen (campo image vacío).
Encuentra todos los productos que SÍ tienen imagen.
Compara cuántos productos hay en cada grupo.

Pistas:
    Campo vacío en Odoo es = False
    Campo no vacío es != False
    Usa search_count() para comparar cantidades

Modelo: ecommerce.producto_api

#Encuentra todos los productos que NO tienen imagen (campo image vacío).
imagen_false = self.env['ecommerce.producto_api'].search([('image', '=', 'false')])
conteo_false = len(imagen_false)

#Encuentra todos los productos que SÍ tienen imagen.
imagen_true = self.env['ecommerce.producto_api'].search([('image', '!=', 'false')])
conteo_true = len(imagen_true)

print(f"La cantidad de productos con imagen es {conteo_true} y sin imagen es {conteo_false}")
